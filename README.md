📘 StudyFlow
StudyFlow is a smart study planner web app designed to help students organize their exam preparation. It generates personalized study plans, tracks progress with a visual progress bar, suggests important topics, recommends books, and allows saving notes for each topic.
🚀 Features
Course & Subject Selection  
Choose your course (Electrical Engineering, Computer Science, etc.) and subject.

Smart Study Plan Generator  
Enter exam date and total topics → StudyFlow calculates topics per day and builds a day-wise plan.

Progress Tracking  
✔ Check off completed topics
📊 Visual progress bar + percentage updates in real time
Topic Suggestions  
Get a curated list of important topics for each subject.

Book Recommendations  
Recommended textbooks and references for deeper learning.

Topic Notes  
📝 Save personal notes for each topic
📖 Auto-loads detailed default notes (like definitions, explanations)
🗑 Delete or overwrite notes anytime

Responsive UI  
Clean, modern design with attractive colors and smooth animations.
🛠 Tech Stack
HTML5 – Semantic structure

CSS3 – Modern styling, gradients, responsive layout

JavaScript (ES6+) – Dynamic functionality, localStorage for notes
